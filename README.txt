关于格式：

1.  Markdown 格式书写，请遵循如下准则。

    加粗用**加粗**，斜体用*斜体*，加粗并斜体用***加粗并斜体***。

    列表的使用如下：

    -   无序列表 item 1
    -   无序列表 item 2

    下面则是有序列表：

    1.  one
    2.  two

    或者这样也是可以的：

    #.  still one
    #.  still two

    列表可以嵌套：

    1.  one
        -   11
        -   12
    2.  two

    缩进永远用 4 个空格，所以不要这样写：

    - 糟糕的无序列表 item 1
    - 糟糕的无序列表 item 2

    更多格式说明见 pandoc 文档：<https://github.com/jgm/pandoc>。

2.  内容折叠

    语法是“-<”和“+<”。前者初始为关闭，后者为打开。都可以 toggle 其中的内容。
    例子如下：

    这个抽屉默认折叠起来 -<

    :   这是抽屉里面的内容。

        内容。

        内容……

        嵌套一个默认打开的折叠抽屉 +<

        :   这是抽屉里面的内容。

            内容。

            内容……

            内层完。

        外层完。

3.  引入其它文件

    在文本中使用 @include <-=path/to/file=

    “@”前面的符号会 prepend 到 included 文件的每一行。
    “-”和“=”之间可以插入额外的字符。

    示例：

    +-----------------------------------------------+
    |                                               |
    |   -   include code                            |
    |                                               |
    |       :   ```cpp                              |
    |           @include <-=2016/democode.cpp=      |
    |           ```                                 |
    |                                               |
    |   -   include excerpts                        |
    |                                               |
    |       @include <-|   =2016/poem.txt=          |
    |                                               |
    +-----------------------------------------------+

    如果 2016/democode.cpp（目录从 repo 根目录起）内容如下

    +------------------------2016/democode.cpp------+
    |                                               |
    |   #include <stdio.h>                          |
    |                                               |
    |   int main() {                                |
    |       printf( "hello world.\n" );             |
    |   }                                           |
    |                                               |
    +-----------------------------------------------+

    +------------------------2016/haizi.txt---------+
    |                                               |
    |   Like tears                                  |
    |   in the rain.                                |
    |                                               |
    +-----------------------------------------------+

    include 后的效果如下：

    +-----------------------------------------------+
    |                                               |
    |   -   include code                            |
    |                                               |
    |       :   ```cpp                              |
    |           #include <stdio.h>                  |
    |                                               |
    |           int main() {                        |
    |               printf( "hello world.\n" );     |
    |           }                                   |
    |           ```                                 |
    |                                               |
    |   -   include excerpts                        |
    |                                               |
    |       |   Like tears                          |
    |       |   in the rain.                        |
    |                                               |
    +-----------------------------------------------+
