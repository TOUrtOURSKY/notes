## 说明

本【笔记系统】提供一些快捷键，包括：

-   `expand` -> 打开所有 folder（依次用键盘输入“e”，“x”，“p”，“a”，“n”，“d”）
-   `<left><left><down>` -> 打开所有 folder（依次按方向键“左”，“左”，“下”）
-   `fold` -> 关闭所有 folder
-   `<left><left><up>` -> 关闭所有 folder
-   `hide` -> 隐藏部分内容（少儿不宜？）
-   `show` -> 打开隐藏的内容
